Problem 01
보행 기반 모델의 한계
CPM 예측모델은 보행 최적화로 개발됨. 좌업·근력운동에서는 CPM–METs 관계가 성립하지 않아 추정 자체가 불가능한 구조적 공백이 존재함.
→ 좌업: CPM 높음 / 실측 1.5 METs | 푸쉬업: CPM 낮음 / 실측 5.5 METs
Problem 02
국가 통계와의 직접 연계
국민건강영양조사 신체활동량 측정조사 3년 연속 수행하며 분석 공백을 직접 확인. 손목 착용 전환 추진 중이나 좌업·근력운동 강도 기준값 전무.
→ 본 연구 결과를 국건영 정책 기준값으로 직접 제공 예정
Problem 03
활동 식별 근거 부재
전용 모델을 개발해도 활동 유형을 자동 식별할 수 없으면 현장 적용 불가. FFT 분석을 통해 활동별 고유 주파수 패턴을 규명해야 함.
→ FFT 패턴 데이터셋 → 향후 머신러닝 활동 자동 분류로 확장
연구책임자 연계 경험
국건영 3년 연속 참여
제9기 3차(2024) · 제10기 1차(2024) · 제10기 2차(2026) 수행 중. 대용량 시계열 가속도 데이터 처리 역량 보유.
→ 현장에서 직접 확인한 분석 공백을 연구 목표로 구체화
H 1
CPM 예측모델 우수성
좌업·근력운동에서 손목 CPM과 실측 에너지소비량 간 활동 유형별 유의한 회귀 관계가 존재하며, 개발된 전용 모델이 기존 보행 기반 모델보다 추정 정확도가 높을 것이다.
H 2
FFT 주파수 고유 패턴
좌업·근력운동의 원시 가속도 신호를 FFT로 분석하면, 활동 유형별로 고유한 주파수 패턴이 나타날 것이다.
